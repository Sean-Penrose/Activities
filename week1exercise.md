### 1. What is the CLI and how to work with it.
**Answer**- CLI stands for Command Line Interface. The command line interface is a way for a user to interact with the computer through the keyboard to take a wide array of actions. Many programs use a CLI such as git and Python. Each program has its own set of specific commands that are typed in the terminal to execute various functions.

### 2. What is a versioning control system and why we need it.
**Answer**- The versioning control system is essentially a way to share, maintain, and back up our code through git and guthub. The main function is to allow the user to be able to have a database for their project's history. The system  works through a set of interactions between Git & Github. 
This helps track changes, go back to previous version, and branch out.
### 3. Understand the differences between Git and GitHub?
**Answer**- Git is the local CLI that you use on your computer and GitHub is the cloud repoitory to store the code you make as well as everyone else who uses the platform.
### 4. Understand the difference between the main branch and other branches of a repository.
**Answer**- The main branch is the finalized code and the other branches are the parts of code that we are working on to be finalized.
